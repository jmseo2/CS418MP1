Operating System: OS X

Compilation & Running
1. In the folder where the main.cpp is, type make in the command line.
2. Run the program by typing ./mp1

Keyboard inputs in the program
- There are three keyboard inputs, 'a', 's', 'd'.
- Pressing 'a' shows/hides the filled polygon. If the polygon is already shown, it will hide it and vice versa.
- Pressing 's' shows/hides the wireframe. If the wireframe is already shown, it will hide it and vice versa.
- Pressing 'd' shows/hides the border of the shape. If the border is already shown, it will hide it and vice versa.
